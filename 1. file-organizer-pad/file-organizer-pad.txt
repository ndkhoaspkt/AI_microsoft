📂 File Organizer Automation (Power Automate Desktop)
📌 Overview
This project is a Power Automate Desktop (PAD) workflow designed to automatically organize files from a selected source directory into categorized subfolders based on file type.
Additionally, it supports batch processing of .txt files by writing indexed content into each file.

🎯 Objectives
Improve daily file management efficiency
Reduce manual sorting tasks
Demonstrate automation skills using Power Automate Desktop

⚙️ Key Features
📁 Dynamic folder selection (source & destination)
🔍 File classification by extension:
PDF → PDF/
Excel (.xlsx) → EXCEL/
Others (e.g., DOCX) → DOCX/
📝 Batch processing of .txt files:
Automatically writes indexed content into each file
🔁 Efficient looping using both:
For Each (file classification)
Loop with index (text processing)

🔄 Workflow
1. Select Folders
User selects:
Source Folder → Origin_Folder
Destination Folder → New_Folder

2. Retrieve Files
Get files in Origin_Folder (*.pdf, *.docx, *.xlsx)

3. File Classification (For Each Loop)
For each CurrentItem in Files
If extension contains pdf
→ Move to: New_Folder\PDF
Else if extension contains xlsx
→ Move to: New_Folder\EXCEL
Else
→ Move to: New_Folder\DOCX

4. TXT File Processing
Get files in Origin_Folder (*.txt)
Loop from 0 to Files.Count - 1

Write content into each file:

"This is file #" + (LoopIndex + 1)

🧠 Technical Highlights
Use of list variables (Files)
Understanding of:
CurrentItem vs Index-based access
File system automation:
Move file operations
Batch file processing
Logical branching (If / Else If / Else)

⚠️ Notes
Ensure destination subfolders exist:
/PDF
/EXCEL
/DOCX
.txt files will be overwritten when writing content
File classification is based on extension string matching
